lookahead(self, table, element, fields, tree=None, dmap=None)

source code 

Performs a look-ahead for referenced resources in both the database and the given tree. Entries are added to the given dmap (demand map), and are returned as list.

Each entry consists of:

  • resource (the resource name)
  • element (the referenced <resource> element in the tree)
  • id (the DB record ID)
  • uid (the UUID of the tree resource)
