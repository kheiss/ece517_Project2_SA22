set_default_timezone(new_tz)

source code 

Sets the default time zone used by the objects contained in this module. new_tz may be either a pytz-compatible tzinfo (requires normalize and localize methods), or a time zone name known to pytz.

_get_fixed_offset_tz(offsetmins)

source code 

For internal use only: Returns a tzinfo with the given fixed offset. This creates only one instance for each offset; the zones are kept in a dictionary
